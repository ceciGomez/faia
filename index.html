Bienvenido

En este sitio intentaremos desarrollar una aplicación que resuelva problemas criptoaritmeticos

Descripción del problema

La criptoaritmetica es un área de la matemática que intenta relacionar simbolos distitntos (generalmente letras) a numero distintos y operar con dichos símbolos (generalmente palabas). De esta manera, puede generarse operaciones no convencionales, como el conocido caso de SEND + MORE = MONEY. Claramente este tipo de problemas genera un gran espacio de estados que son potenciales soluciones del problema. Si bien existen varias alternativas para resolver estos problemas (métodos inductivos y deductivos), existen muchas variantes de algoritmos de enjambre (bees, birds, ants, etc) que pueden ser utilizados para resolver problemas de optimización. Este trabajo consiste en proponer un método basado en algoritmos de enjambre que resuelva problemas criptoaritmeticos en general, con operaciones de suma y resta con operandos variables (cantidad y tamaño). Se pretende que la aplicación desarrollada tenga la funcionalidad necesaria para manejar al menos tres operandos de cinco dígitos cada uno con las mencionadas operaciones.
